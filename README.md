
LemonThings
=======

LemonThings — это мод для Minecraft 1.21.1 на NeoForge, который добавляет множество полезных, интересных и хорошо сбалансированных предметов, органично вписывающихся в ванильную игру.

Мод расширяет возможности выживания, крафта и исследования, не нарушая при этом атмосферу Minecraft.
